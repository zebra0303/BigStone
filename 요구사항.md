# BigStone Task Manager

이 프로그램을 오늘 중요하게 할일을 등록해서 잊지 않게 해주는 Todo 웹앱입니다.
항아리에 큰돌 부터 넣어야 작은돌을 넣을 수 있습니다. 즉 중요한 일부터 처리해야 작은일들을 처리할 수 있습니다.

할일은 2가지 종류가 있습니다.

1. 오늘 끝내야 할일
2. 기간을 정해서 기일 내에 끝내야 할일

할일 목록 페이지 상단에는 오늘 끝내야 할일 목록을 리스트 업하고.
그 아래는 기간을 정해서 기일 내에 끝내야 할일 목록을 리스트 업합니다.
할일 목록중에 중요한 할일은 아이콘과 주요색상(관리자 화면에서 설정 가능)으로 표시 합니다.

### 할일에 입력할 항목

1. 할일 제목 (필수)
2. 할일 내용 (선택)
3. 중요항목 (체크박스)
4. 할일 기한 (달력 선택 - 기본 오늘)
5. 할일 상태 (드롭다운 : 할일, 진행중, 완료)
6. 반복일정 (드롭다운 : 없음, 매일, 매주, 매월, 매년)
   - 매주 : 요일 선택
   - 매월 : 일 선택
   - 매년 : 월, 일 선택
7. 알림설정 (드롭다운 : 없음, 10분전, 30분전, 1시간전, 2시간전, 4시간전, 8시간전, 12시간전, 24시간전)

#### 완료 항목 관리

- 오늘 완료한 항목은 리스트에서는 표시 됩니다.
- 오늘 이전 완료한 항목은 따로 보관하고 필요시 검색 및 조회가 가능합니다.

### 관리자 옵션

- 주요색상 설정
- 알림설정
  - 알림시간 설정
  - 슬랙 웹훅(옵션)
